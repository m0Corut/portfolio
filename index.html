01. Hakkımda

Bartın Üniversitesi Bilgisayar Mühendisliği 5. sınıf öğrencisiyim. 2 yıldır web geliştirme alanında aktif olarak çalışıyor, modern teknolojilerle full-stack uygulamalar geliştiriyorum.

React, Next.js, Node.js, TypeScript gibi teknolojilerle kurumsal seviyede projeler geliştirdim. Backend geliştirme, veritabanı yönetimi ve RESTful API tasarımı konularında deneyimliyim. 2024 yılında Google Developer Student Clubs (GDSC) Bartın yönetim ekibinde yer aldım.
